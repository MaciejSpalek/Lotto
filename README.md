## About the project 
The Application based on pseudo-randomness.
Click in the first button and compare your drawn numbers with numbers from official drawing. 

## The project created with 
 * HTML
 * CSS + SASS
 * JS + jQuery
 * Responsive Web Design
 * Working with API
